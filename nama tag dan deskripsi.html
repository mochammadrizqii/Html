Tag Deskripsi
<!--...--> Menentukan komentar
<!DOCTYPE> Menentukan jenis dokumen
<a> Menentukan hyperlink
<abbr> Mendefinisikan singkatan atau akronim
<address> Menentukan informasi kontak untuk penulis / pemilik dokumen
<area> Menentukan area di dalam peta-gambar
<article> Mendefinisikan sebuah artikel
<aside> Defines content aside from the page content
<audio> Menentukan konten suara
<b> Menentukan teks tebal
<base> Menentukan URL dasar / target untuk semua URL relatif dalam sebuah dokumen
<bdi> Mengisolasi bagian teks yang mungkin diformat dalam arah yang berbeda dari teks lain di luarnya
<bdo> Mengganti arah teks saat ini
<blockquote> Menentukan bagian yang dikutip dari sumber lain
<body> Menentukan isi dokumen
<br> Menentukan jeda baris tunggal
<button> Menentukan tombol yang dapat diklik
<canvas> Digunakan untuk menggambar grafik, dengan cepat, melalui scripting (biasanya JavaScript)
<caption> Menentukan keterangan tabel
<cite> Menentukan judul sebuah karya
<code> Mendefinisikan sepotong kode komputer
<col> Menentukan properti kolom untuk setiap kolom dalam elemen (colgroup)
<colgroup> Menentukan grup yang terdiri dari satu atau beberapa kolom dalam tabel untuk pemformatan
<data> Tautan konten yang diberikan dengan terjemahan yang dapat dibaca mesin
<datalist> Menentukan daftar opsi yang telah ditentukan untuk kontrol input
<dd> Menentukan deskripsi / nilai istilah dalam daftar deskripsi
<del> Menentukan teks yang telah dihapus dari dokumen
<detail> Menentukan detail tambahan yang dapat dilihat atau disembunyikan pengguna
<dfn> Merupakan instance dari suatu istilah
<dialog> Menentukan kotak dialog atau jendela
<div> Menentukan bagian dalam dokumen
<dl> Menentukan daftar deskripsi
<dt> Menentukan istilah / nama dalam daftar deskripsi
<em> Menentukan teks yang ditekankan
<embed> Defines a container for an external (non-HTML) application
<fieldset> Grup terkait elemen dalam formulir
<figcaption> Menentukan keterangan untuk elemen (figure)
<figure> Menentukan konten mandiri
<footer> Menentukan footer untuk dokumen atau bagian
<form> Menentukan formulir HTML untuk input pengguna
<h1> to <h6> Menentukan judul HTML
<head> Menentukan informasi tentang dokumen
<header> Menentukan header untuk dokumen atau bagian
<hr> Mendefinisikan perubahan tematis dalam konten
<html> Menentukan akar dari dokumen HTML
<i> Menentukan bagian teks dengan suara atau suasana hati alternatif
<iframe> Menentukan bingkai inline
<img> Menentukan gambar
<input> Menentukan kontrol input
<ins> Menentukan teks yang telah dimasukkan ke dalam dokumen
<kbd> Menentukan input keyboard
<label> Menentukan label untuk elemen (input)
<legend> Menentukan keterangan untuk elemen (fieldset)
<li> Menentukan item daftar
<link> Menentukan hubungan antara dokumen dan sumber daya eksternal (paling sering digunakan untuk menautkan ke style sheet)
<main> Menentukan konten utama dokumen
<map> Menentukan peta gambar sisi klien
<mark> Menentukan teks yang ditandai / disorot
<meta> Menentukan metadata tentang dokumen HTML
<meter> Menentukan pengukuran skalar dalam rentang yang diketahui (pengukur)
<nav> Menentukan tautan navigasi
